## cd (change directory)

- 현재 작업중인 디렉토리(=폴더)를 변경하겠다.
- 앞에 몇단어 치고 탭 눌러서 자동완성 사용하면 편함

``` bash
Bakpak@LG-Gram MINGW64 ~/Desktop/github_study
$cd

Bakpak@LG-Gram MINGW64 ~
$ cd Desktop/***/**** {directory name}

Bakpak@LG-Gram MINGW64 ~/Desktop/github_study
$cd .. # 상위 폴더로 이동

Bakpak@LG-Gram MINGW64 ~/Desktop/github_study
$cd pwd  # 현재 위치 확인

```



## mkdir (make directory)

